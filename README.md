# desktop-audio-cast

Based on [mafintosh@chromecasts](https://github.com/mafintosh/chromecasts).
A tiny interface in Node to control Google cast devices.

```
npm install
node index
```

## Link

To use along with [audio-streamer] (https://github.com/L4ngu0r/audio-streamer)
